# Data Structures and Algorithms (DSA) in C

This repository contains my personal DSA notes and implementations in **C**, written to understand each concept clearly.  
Each folder includes explanations, dry runs, and working code for common data structures and algorithms.

  - Insertion, Deletion, Searching

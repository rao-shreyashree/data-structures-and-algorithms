# Heap Tree

This folder contains implementation of **Heap Tree** in C.

---

## Overview
A **Heap Tree** is a type of binary tree with keys assigned to its nodes (one key per node). Following two conditions should be met: 
1. tree shape: all levels except last levek, are essentially complete 
2. parental dominance: key at each node is greater than or equal to its children 


---

## How to Use
Navigate and compile any `.c` implementation file:
```bash
gcc file_name.c -o file
file
```
